# Expense Manager

A beginner project expense management web application built using HTML, CSS, and JavaScript.  
The project focuses on practicing core JavaScript logic, data handling, and building small applications end-to-end.

## Features
- Add, edit, and delete expenses
- View total and category-wise expenses
- Visualize spending patterns using charts
- Persist data using browser LocalStorage
- Filter expenses by category

## Technologies Used
- HTML
- CSS
- JavaScript
- Chart.js

## Project Structure
├── index.html  
├── style.css  
├── app.js  
├── data.js  
├── ui.js  
├── chart.js  
├── README.md  

## Why this project
This project was built to strengthen JavaScript fundamentals, practice logic building, and understand how client-side applications manage and persist data.
